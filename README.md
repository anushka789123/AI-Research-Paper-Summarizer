# AI Research Paper Summarizer 

AI Research Paper Summarizer is an AI-powered application that helps users quickly
understand research papers by generating concise summaries from PDF documents.
The system uses Natural Language Processing (NLP) and a
Retrieval-Augmented Generation (RAG) pipeline to produce context-aware summaries.

## Features
- Upload research paper PDFs
- Automatic text extraction from PDF files
- Context-aware summarization using RAG
- Vector-based document retrieval
- Interactive Streamlit web interface

## Tech Stack
- Python
- Streamlit
- LangChain
- FAISS
- Sentence Transformers
- NLP & Embeddings

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
